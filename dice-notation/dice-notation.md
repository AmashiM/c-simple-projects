

## Tokenizer

| symbol | identity |
| :----: | :------: |
| d | dice |
| k | keep operation |
| l | usually following after a k operation, defines a keep lowest operation |
| h | usually following after a k operation, defines a keep highest operation |
| + | addition |
| - | subtraction |
| * | multiplication |
| / | division |
| 0-9 | number |
| () | parenthesis used to group operations |
